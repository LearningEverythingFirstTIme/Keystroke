How it Works

Four steps. Zero config.

Download, enter your API key, start typing. Suggestions appear near your cursor in any application.

01

Watch you type

A system-wide keyboard hook captures keystrokes in every app. Text stays in memory — never written to disk.

02

Read the screen

Optional OCR captures visible text for context — the conversation, document, or code you're looking at.

03

Predict what's next

Your typed text + screen context is sent to your chosen AI engine. Streaming responses appear in ~150ms.

04

Type it for you

Press Tab to accept the full suggestion, or Ctrl+Right to accept one word at a time. Esc to dismiss.

Features

Built different.

Not another browser extension. A native Windows app that works everywhere.

⚡

Streaming predictions

Suggestions appear progressively as the AI responds. First words in ~150ms — you see it before it's done thinking.

🖥

OCR screen reading

Captures visible text using Windows built-in OCR. Knows what conversation you're replying to, not just what you're typing.

🧠

App-aware tone

Casual in Discord, professional in Outlook, precise in VS Code. Automatically adjusts based on the active app.

🔄

Multi-suggestion cycling

Not sure about the first suggestion? Ctrl+Up/Down to browse alternatives loaded in the background.

✂️

Word-by-word acceptance

Press Ctrl+Right or Shift+Tab to accept just the next word. Full control over how much you take.

🛡

PII redaction

Credit cards, SSNs, emails, and phone numbers are automatically stripped before anything leaves your machine.

🎯

Smart debounce

Instant prediction at word boundaries. Fast 100ms debounce mid-word. No waiting, no lag.

📚

Rolling context

Remembers your last 500 characters of accepted text for continuity across multiple completions in a row.

🎓

Learns your style

Opt-in learning uses your accepted completions as few-shot examples. Gets better at matching how you write.

Trust

Your keyboard. Your data.

A keyboard hook app is only as good as its privacy story. Here's ours.

🔍 Fully open source

Every line of code is on GitHub. Audit it yourself. Build from source. No secrets, no obfuscation.

🔐 Encrypted API keys

Your credentials are encrypted with Windows DPAPI. Only your Windows account can decrypt them. Never transmitted to anyone but your chosen AI provider.

🚫 No telemetry

Zero analytics, tracking, crash reporting, or update pinging. The only network connections are HTTPS to your AI provider.

📝 Transparent privacy policy

Detailed documentation of exactly what data is collected, where it goes, and how to delete it all. No legalese.

🛑 Always your choice

Ctrl+Shift+K to toggle off instantly. Disable OCR, disable learning, or close the app. You're always in control.

💾 Local only

No cloud storage, no user accounts, no server-side component. All data lives in %AppData% on your machine.
