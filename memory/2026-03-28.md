### Key Learnings
1. Gemini 2.5 Flash has "thinking" tokens that consume output budget — must disable with `thinkingBudget: 0`
2. Tab key must be swallowed via hook return value `(IntPtr)1` or it inserts tab char
3. Panel must NOT steal focus (`WS_EX_NOACTIVATE`) or SendInput types into wrong window
4. `GetCaretPos` doesn't work cross-process — mouse position is the practical fallback
5. WPF `ShutdownMode="OnExplicitShutdown"` keeps app alive without visible window
